# Mock Photo Sources

These generated images are used only for Look App Store / demo screenshot
generation. They are not real user library photos.

- `realistic_photo_contact_sheet.png` is the generated 4x4 source sheet.
- `demo-photo-01.png` through `demo-photo-16.png` are the cropped source tiles.
- `demo_photo_asset_contact_sheet.png` is a review sheet for the cropped tiles.

The iOS screenshot host consumes the same tiles through
`ios/Look/Assets.xcassets/DemoPhotos`.
